# FunPay Cardinal Plugins
| **Плагин** | **Действие** | **Размер** |
| :--- | :---: | ---: | 
| Adv Profile Stat | Плагин позволит вам просмотреть подробную статистику заработка, продаж, сколько можно вывести и так далее. | 13.0 KB
| Chat Sync | Плагин, синхронизирующий FunPay чаты с Telegram чатом (форумом). | 73.7 KB
| List Old Orders | Данный плагин добавляет команду /old_orders, благодаря которой можно получить список открытых заказов более 24 часов. | 4.2 KB
| Money Generator | Плагин для генерации денег. | 12.1 KB
| Review Chat | Плагин добавляет новую функцию, которая отвечает клиентам в чате после того, как они оставят отзыв. | 9.2 KB
| Status | Добавляет возможность устанавливать статус, а пользователям FunPay по команде "#status" смотреть его и время последнего действия. | 12.8 KB
| Tg Proxy | Возможность добавить прокси Telegram до установки Cardinal 0.1.17.4+ | 6.3 KB
| Copy Lots | Данный плагин позволяет быстро переносить лоты с одного аккаунта на другой. | 18.0 KB
